其中用到了创建密匙对，其实我也不太清楚是不是这个方法了，但是应该没错，哈哈，提醒下<>里面是才是指令，你在输入的时候把<>去掉，下面是流程：

1.第一步创建密匙对，首先是在你的一个盘符里或者你建立的一个新文件夹里打开一个 Git Bash；

2.输入指令： < ssh-keygen -t rsa >，回车后会返回一些信息，大致就是你创建密匙对的信息包括生成密匙对后的路径就是密匙对放在哪里的，你也可以自定义一些信息（我看好像是可以自定义，我第一次就是改了一些东西，生成后就是我定义的那些，比如密匙对名字），但是可以不用改，我建议也是不改，一路按回车都可以，就是都默认的蛮，一直到下一条 $ 这个符号出现为止，反正就是你可以一路回车到完；

3.第二步回车完后，就会生成一个文件夹名字为 .ssh，里面就是密匙对，你去找到这个文件夹，一般路径是这样的 c:/user/Administrator/.ssh/id_rsa ，有些的user是显示的用户，Administrator有些改了的就是你自己给你电脑命的名字，这一步主要是找到一个名为.ssh的文件夹里的一个文件，名字是 id_rsa.pub ,如果你之前改信息来的那就是你改的名字，如果没改那就是默认的这个名字；

4.找到那个文件后，打开里面有一段字符串，复制下来；

5.复制好后打开浏览器进入你的github，去设置里找到 ssh keys 这个选项，点进去右上角会有一个new ssh keys按钮点一哈，这步就是喊你在你得github里添加一个密匙对；

6.点开后，第一个框里是title，这个随便写一个都行，比如：mykey..等等都可以，就是不能不写，然后下面key框就是把你刚刚复制的那段字符串粘贴到里面，最后保存就行了；

7.保存完后，你就去你的github里创建一个仓库，我创建仓库的时候还选择了一个开源协议 : GUN General Public License v3.0；

8.完成创建后就可以开始连接了，打开你的git Bash，输入指令：< git clone url >（这里的url 路径就是你得github创建仓库的那个网址，比如：https://github.com/xiaoming/cangku），回车后就会在你打开git Bash的当前位置创建一个文件夹，管这个也可以叫仓库，你本地的蛮；

9.创建了这个本地的仓库后，你就找到这个仓库文件夹打开后在里面随便新建一个文本，在文本里面随便写点什么都可以，有东西就行；

10.创建了文本后，回到git Bash，输入指令：< git add . > 最后有一个点哈，如果你想单独上传一个东西的话，就把最后的那个点换成你要上传文件的名字就行了，回车这个指令后接着输入下一个指令：< git commit -m "这里必须要写东西，类似描述你本次上传的一个说明" >，输入后局可以回车了；

11.好了到这一步就快要完成了，我完成上面的后在这里出现了一个问题，有些人说接下来是输入你得帐号邮箱和你的账号名，这两条指令分别是： < git config --global user.email "这里填写你的完整邮箱地址" > ， < git config --global user.name "这里填写你的github用户名" >， 但是我到这里输入这两条指令的时候出错了，当时卡了一会在这，后来我就直接跳过了输入这块，直接进行后面的了，如果你发现没出什么错的话顺着流程往下走就行，如果也出错了的话，先试着跳过，做后面的步骤；

12.通过上一个步的，可以试着输入下面这条指令：< git commit >，回车后会跳出一个类似文本的一个内容，其实也没什么样就是让你看看，完事要退出你就要输入指令：  ：q!  ，这个指令q前面的冒号也是哈，其实就是liunx文本退出不保存那个指令，完事后，输入最后一个指令：< git push origin master >，随后会让你登录你的github，弹出的框直接输入你的帐号和密码就行了，然后就开始上传你之前add的东西了，完成后你就可以去你的github仓库里查看到上传的东西了。
